GitHub Note:
============

New Repo:
1. Step: git init
2. Step: git add .
3. Step: git commit -m "Comment goes here"
4. Step: git git branch -M main
5. Step: setup remote origin like this => git remote add origin https://github.com/softsamiul/first-repo.git 
6. Step: git push -u origin main

After modification we have to do 3 things:
------------------------------------------

1. Step: git add
2. step: git commit
3. Step: git push 

Git branching command:
----------------------
To see branches => git branch
To create new branch => git branch branch-name
To switch branch => git checkout branch-name
To merge changes => git merge branch-name
